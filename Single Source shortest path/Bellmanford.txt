Bellman-Ford(G,W,s)
Initialize_Single_Source(G,s)
for i <- 1 to |G.V| - 1
    for each edge (u,v) E G.E 
        Relax(u,v,w)
for each edge (u,v) E G.E 
    do if v.d > u.d + w(u,v)
        return false
return true

// Time complexity: Best, Average - O(|E||V|) ~ O(n^2)
//                 Worst - O(n^3)
